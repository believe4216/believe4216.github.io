Markdown中设置图片的对齐方式

Markdown 图片插入语法功能很单一，无法设置图片的对齐方式。上网搜索一下解决办法，发现可以通过在 Markdown文件中嵌入 HTML 语句的方式来设置图片的格式。

其中的<img>语句还可以设置图片大小。
